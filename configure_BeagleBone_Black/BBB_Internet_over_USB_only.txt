## BeagleBone internet over USB only

For more information see:  
[http://robotic-controls.com/learn/beaglebone/beaglebone-internet-over-usb-only ](http://robotic-controls.com/learn/beaglebone/beaglebone-internet-over-usb-only)  

1. On the BeagleBone, write in a terminal:
user@bbb:~$ sudo ifconfig usb0 192.168.7.2  
user@bbb:~$ sudo route add default gw 192.168.7.1

2. On the Linux computer, write in a terminal:
#eth0 is my internet facing interface, eth1 is the BeagleBone USB connection.
user@pc:~% sudo su
root@pc:~% ifconfig eth1 192.168.7.1
root@pc:~% iptables --table nat --append POSTROUTING --out-interface eth0 -j MASQUERADE
root@pc:~% iptables --append FORWARD --in-interface eth1 -j ACCEPT
root@pc:~% echo 1 > /proc/sys/net/ipv4/ip_forward

3. On the BeagleBone, write in a terminal:
user@bbb:~% cat /etc/resolv.conf

# Dynamic resolv.conf(5) file for glibc resolver(3) generated by resolvconf(8)
#     DO NOT EDIT THIS FILE BY HAND -- YOUR CHANGES WILL BE OVERWRITTEN
domain localdomain
search localdomain
nameserver 192.168.1.1

4. On the BeagleBone, in the file /etc/resolv.conf add the line 

nameserver 8.8.8.8

This file should be: 

# Dynamic resolv.conf(5) file for glibc resolver(3) generated by resolvconf(8)  
# DO NOT EDIT THIS FILE BY HAND -- YOUR CHANGES WILL BE OVERWRITTEN

search localdomain
nameserver 8.8.8.8
nameserver 192.168.1.1
